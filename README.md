# Plasmodium R Statistical Analysis

This repository contains a university data analysis project carried out with R.  
The project focuses on biological data related to *Plasmodium* and mosquito vectors.

The goal was to practice a complete data analysis workflow, from data preparation and visualization to statistical modelling and interpretation of results.

## Project Background

In this project, biological measurements were analyzed to investigate how *Plasmodium* responds after exposure to different mosquito vectors.

The analysis was part of a Data Mining course and helped me apply statistical methods to real biological data.

## What I did

- prepared and checked biological measurement data
- created visualizations with R
- compared response patterns between different groups
- used statistical models to analyze the data
- interpreted the results in a biological context

## Repository Structure

```text
methods/          R scripts used for the analysis
results/figures/  Selected plots and visualizations
results/tables/   Selected result tables or model outputs
docs/             Short project notes and documentation
```

## Data Availability

The original dataset is not included in this repository because it was provided in the context of a university course project and is not publicly shared.

For this reason, the repository focuses on the analysis structure, R code, statistical methods and selected non-sensitive results.
